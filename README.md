# Team Task Manager

A Django-based web application for managing projects and tasks with role-based access.

## Features

- Authentication (Login/Signup)
- Project Management
- Task Management
- Dashboard Analytics
- REST APIs
- Admin & Member Roles

## Tech Stack

- Python
- Django
- Django REST Framework
- SQLite
- Bootstrap

## APIs

- /api/tasks/
- /api/projects/
